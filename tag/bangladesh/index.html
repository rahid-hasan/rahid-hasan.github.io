MozIftar Dhaka 2013

Standard

In the last year Mozilla Bangladesh first time arranged MozIftar in Dhaka. It was an interesting event. Iftar basically an Islamic term. In the holy month of Ramadan muslim people fasting the whole day & take meal at the dawn. It is called Iftar. We mozillians from Bangladesh do that thing together with Mozilla mission. And named it MozIftar. From last year we are regularly arranging MozIftar in Bangladesh. This event taken place in Star Kabab (Old Dhaka) on July 29, 2013.

We had some agendas for this meetup:

  • Set a plan & direction for Mozilla Bangladesh for next one year.
  • Build a event calendar.
  • If possible create some program specific task force.
  • Discuss about event plan (cost – benefit & event target).
  • We have some issues to discuss face to face.
  • And ofcourse Iftar :-)
